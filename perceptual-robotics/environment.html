How to run code?
  1. Open any python codes in Demos folder
  2. The system will open Geany (Integrated Development Environment, IDE) shown as below:


  3. You can click the this button in the toolbar or click -> Build -> Execute (F5 is hotkey) in manubar to execute the python code. (The system will run the code with python2 by default. )




How to specitify python3 to run code?
Since some demo codes need advanced libaries that only work with python3, it's necessary to set python3 to execute the corresponding codes. The steps are as follows:
  1. click ->Build -> Set Build Commands in manubar and set "ExecutePy3" in Execute commands as below:


  2. Then check the newly added item "ExecutePy3" under Build menu.
