Space-elevaotr: 1- DOF Mass-Spring-Damper System

Introduction
In this tutorial, we are going to build a 1-dof mass-spring-damper system with LEGOs. The viscous friction between the load and the tower is small.

Sensor Installation

To measure the oscillation of the cart, we can install a proximity sensor on the side of the platform. We use glue to fix the sensor to the Tile.

We mount the sensor module on the base plate below the load.



Rubber Band And Lines Installation

To make the load vibrate, we can install a rubber band above the load. At the same time, we use four lines to limit the movement of the load in other than vertical directions.
